Python：学习笔记

发表于 2016-02-18 | 更新于 2016-02-20 | 分类于 Python |

字符串的截取

  • 当使用以冒号分隔的字符串，结果包含了以这对偏移标识的连续的内容，左边的开始是包含了下边界，而取到的最大范围不包括上边界
  • [1:] 截取第二个字符到结尾
  • [:5] 截取从头开始到第五个字符之前

条件语句

  • 条件语句后面要写冒号
  • elif=else if
  • 字符串中的空格也可被in语句选取

循环语句

  • for语句相当于给num重新定义

  • 循环语句后面要写冒号

  • range语句倒着数则步长前加负号

Tips

  • 文档字符串与注释的区别：文档字符串重在说明我在算什么，注释重在说明怎么算
  • 三个双引号可使字符串换行
  • 删除list中的空字符：new_list = [ x for x in list if x != ‘ ‘ ]
